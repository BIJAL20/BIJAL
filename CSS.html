Question 4:


(4) Name a few CSS style components

*Borders: CSS allows you to define borders around elements, specifying properties like border width, border color, and border style (solid, dashed, etc.).
You can control borders for various elements, such as divs, images, or buttons.

* Shadows:: CSS provides properties for creating shadows, including box shadows and text shadows.
Box shadows allow you to add shadows around elements, while text shadows enable you to apply shadows to text, creating depth and visual effects.

* Shadows:: CSS allows you to create gradients, which are smooth transitions between two or more colors.
You can define linear gradients (top to bottom, left to right) or radial gradients (from the center outward), adding visual depth and richness to backgrounds or elements.

  • These are just a few examples of CSS style components that enable you to customize the appearance and behavior of elements on web pages.
  • CSS provides a wide range of properties and features to create visually appealing and interactive designs.

Question 5: What do you understand by CSS opacity?

Answer:

The opacity CSS property controls an element's opacity. The inverse of transparency, opacity, relates to how much of the content behind an element is obscured. Opacity values are always floating numbers, with a minimum of 0 and a maximum of 1. When you use the opacity property to give transparency to an element's background, all of its child elements become translucent as well.

Example:

<p style="width:200px; height: 200px; border: 1px solid black; background-color: rgba(138, 194, 54,);"></p>

(6)How can the background color of an element be changed?

To change the background color of an element using CSS, you can use the background-color property.
This property allows you to set a specific color value for the background of an element.

    Example .element { background-color: red; }
    In the above example, the element with the class "element" will have a background color of red.

Question 13:


(13)What are the ways to integrate CSS as a web page?

There are three methods to integrate CSS on web pages.

1. Inline - Inline CSS involves adding CSS directly within the HTML elements using the style attribute.

2. Embedded/Internal -Internal CSS is placed within the <style> tags in the <head> section of an HTML document. It applies to the entire document or a specific section enclosed within a tag.

3. Linked/Imported/External - This CSS involves creating a separate CSS file with a .css extension and linking it to the HTML file using the <link> element.

Question 14:


(14) What is embedded style sheets?

Embedded style sheets, also known as internal style sheets, are a method of including CSS styles directly within an HTML document. With embedded style sheets, the CSS code is placed within the <style> tags in the <head> section of the HTML file.

    Example:

    <style>
    h2{color:blue}
    </style>

Question 15:


(15) What are the external style sheets?

An external style sheet is a separate CSS file that can be accessed by creating a link within the head section of the webpage.

External CSS is used to apply CSS on multiple pages or all pages. Here, we write all the CSS code in a css file. Its extension must be .css for example style.css.

    Example:

    p{color:pink}

    You need to link this style.css file to your html pages like this:

    <link rel="stylesheet" type="text/css" href="style.css">

(16) What are the advantages and disadvantages of using external style sheets?

Advantage of external style sheet:-

  • The external style sheet is generally used when you want to make changes on multiple pages.

    • Disadvantages of External CSS:

  • Your pages may not be rendered correctly until the external CSS is loaded.
